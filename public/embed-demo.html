TrackStack Embed Mode Demo

This page demonstrates how to embed TrackStack in different configurations for dashboards and iframes.

1. Full Embed Mode (Default)

Shows the complete status view with title and tabs.

URL: /embed

2. Embed with Active Services Only

Shows only active services (no disruptions).

URL: /embed?tab=active

3. Embed with Paused Services Only

Shows only services with disruptions.

URL: /embed?tab=paused

4. Minimal Embed (No Title, No Tabs)

Ultra-minimal view for dashboard widgets.

URL: /embed?hideTitle=true&hideTabs=true

5. Compact Dashboard Widget

Shows only active services with minimal UI.

URL: /embed?tab=active&hideTitle=true&hideTabs=true

Implementation Examples

HTML iframe:

<iframe src="/embed?tab=active&hideTitle=true&hideTabs=true" width="100%" height="300" frameborder="0" title="Tube Status"> </iframe>

React Component:

const TubeStatusWidget = () => ( <iframe src="/embed?tab=active&hideTitle=true&hideTabs=true" width="100%" height="300" frameBorder="0" title="Tube Status" /> );

Available URL Parameters:

  • tab: "all", "active", or "paused" - Filter which services to show
  • hideTitle: "true" - Hide the "Status" heading
  • hideTabs: "true" - Hide the tab navigation
